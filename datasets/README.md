## This directory contains tabular data used throughout the project:

#### -Training datasets used to train Electrolyte-GPT models

#### -Generated datasets produced by sampling scripts during unconditioned or property-conditioned generation

&nbsp;

Typical columns you may see (varies by task):

-SMILES / line notation

-Model temperature

-Conditioning targets (e.g., thresholds for conductivity, oxidative stability, CE)

-ML-predicted properties, validity, uniqueness, novelty markers
